# HW4_SSH
## Задание 1.

Условие:
Переделать все шаги позитивных тестов на выполнение по SSH. Проверить работу. 
Вместо использования других протоколов или подходов, мы будем использовать SSH для выполнения всех необходимых действий.

Условие:
